
*Eye-Blink* - Simple Eyes system for paintball loader

h1. Introduction

This project is designed for paintball loaders, which do not have eyes system. It can reduce consumption of battery power and reduce motor noise by turning motor off if marker is not shooting.

h2. Project files

* README.textile - this file
* hardware - directory for hardware part
* software - directory for software part
* COPING - text of GPL licence

h3. Hardware

The project is designed for AVR microcontroller but it can be adjusted for any other chip.
